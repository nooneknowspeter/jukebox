# jukebox

A collection of themes inspired by music cover art.

The original concept was derived from
[metalelf0](https://github.com/metalelf0/black-metal-theme-neovim),
a collection of theme inspired by black metal bands' iconic cover art.

I have been using the
[dark funeral theme](https://github.com/metalelf0/base16-black-metal-scheme/blob/master/black-metal-dark-funeral.yaml)
for a long time and I've been loving it.

I wanted to expand on the idea,
not just by having a clean and centralized place to have
these themes available for different mediums,
but to have themes for different cover arts of different genres and artists.
Get closer to the feeling of the music through these themes.
