All contributions are welcomed,
we are trying to make something not just beneficial but fun
for the community.

It would mean a lot helping out with `[TODO.md](./TODO.md)`,
issues and discussions pointed out by other users and contributors.

For nix users,
a `[flake.nix](./flake.nix)` is provided in the project,
with a simple command; `nix develop`, you will launch the
development shell with all the project required dependencies
to start developing.
A `[.evnrc](./.envrc)` is also provided.

## Formatting and Style Conventions

Ensure all code is formatted using `treefmt`,
the list of formatters can be found in the `[flake.nix](./flake.nix)`.
Although, it is recommended to use `treefmt`, the formatters listed
in the `[flake.nix](./flake.nix)` can be installed and ran individually.
