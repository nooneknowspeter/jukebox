# jukebox

create themes from music cover art.

## structure

- **manifests**: `src/themes/{artist}/{album}/theme.yaml` - yaml with cover art refs + base16 palette + neovim black-metal fields
- **templates**: `src/jukebox/templates/{category}/{target}.j2` - jinja2 templates per output format
- **output**: `output/{category}/{target}/{artist}-{album}.{ext}` - generated files, committed for others to consume
- **cli**: `main.py` entry point -> `src/jukebox/cli.py`
- **session logs**: `.agent/session-{date}.md` - per-session activity logs for continuity across conversations

## adding a theme

1. create `src/themes/{artist}/{album}/theme.yaml`
2. fill in: `name`, `artist`, `type` (album/single/ep/compilation), `year`, `link` (cover/spotify/wikipedia), `palette` (base00-base0F)
3. run `just generate-all` to produce all output formats
4. submit a pr

## adding a template (new output target)

1. add `src/jukebox/templates/{category}/{new_target}.j2`
2. register it in `_discoverTargets()` in `src/jukebox/generator.py`
3. add the file extension in `EXTENSION_MAP` in `generator.py`

## commands

- `python main.py list` -- list all themes
- `python main.py show <slug>` -- show theme details
- `python main.py generate` -- generate all targets for all themes
- `python main.py generate --target ghostty` -- single target
- `python main.py cover <slug>` -- download cover art
- `python main.py screenshot` -- generate composite images (cover + palette)
- `just format` -- format code
- `just lint` -- lint
- `uv run pytest` -- run tests

## important

- do not commit. stage, review, and let the contributor commit by hand.
- tests should be written by hand, not generated by agents

## conventions

- variables: `lower_snake_case`
- functions: `camelCase`
- constants: `UPPER_SNAKE_CASE`
- classes: `PascalCase`
- trailing commas in multi-line argument lists
- newline before `return`, `raise`, nested clauses
- newline before `# comment`, none after
- all comments lowercase
- `regex` instead of `re`
- `str | None` instead of `Optional[str]`
