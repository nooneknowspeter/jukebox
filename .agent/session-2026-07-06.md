## Project Goal

Transform base16 color palettes extracted from metal album cover art into themed config files for terminal emulators, editors, and other tools - using jinja2 templates.

## Decisions Made

### Architecture

- **Package structure**: code in `src/jukebox/`, themes in `src/themes/`, templates in `src/jukebox/templates/`, schema in `schemas/`
- **`src/themes/__init__.py`** added so setuptools discovers it as a package (included via `packages.find` in pyproject.toml)
- **Bundled themes**: yaml files inside the installed package; `resolveThemesDirectory()` falls back: cli flag → env var → bundled (site-packages) → `~/.local/share/jukebox/themes/`
- **Output dir committed** (not gitignored) so users can consume generated files directly from the repo
- **`release_type`** used in `Theme` dataclass to avoid shadowing python builtin `type`; yaml key stays `type`

### Code Style

- `camelCase` for functions, `lower_snake_case` for variables, `UPPER_SNAKE_CASE` for constants, `PascalCase` for classes
- Trailing commas in multi-line arg lists
- Newline before `return`, `raise`, nested clauses, `# comment` (none after)
- All comments lowercase
- `regex` not `re`; `str | None` not `Optional[str]`

### Nix Flake

- `jukeboxPackage` in `let` block, `packages.default` + `apps.default` below `devShells`
- Not a flake output for nixpkgs submission - just local dev convenience

### Theme Data

- All 17 themes updated: `# yaml-language-server` schema ref, `type: album`, `year`, `author` cleaned up
- metalelf0-derived themes keep `author.name: "metalelf0"`; pallbearer credited to `nooneknowspeter`
- Year corrections queried via websearch: gorgoroth (2003), khold (2002), taake (2005), thyrfing (1999), windir (2001)

### Templates

- **ghostty**: maps base16 palette to 16 color slots in `.conf` format
- **wezterm**: maps to ansi + brights arrays in `.toml`
- **yazi**: categorized as terminal, not gui
- New targets need registration in `_discoverTargets()` + `EXTENSION_MAP` in `generator.py`

## How to Run

```sh
# activate env
uv sync

# list themes
python main.py list

# show theme
python main.py show bathory-hammerheart

# generate all output formats
python main.py generate

# generate single target
python main.py generate --target ghostty

# download cover art
python main.py cover bathory-hammerheart

# nix (if using flake)
nix run .# -- list
```
